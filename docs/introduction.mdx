# Aria

**Aria** is the research-first intelligence interface for decision-makers. Today it is a production-grade chat and deep-research product. Tomorrow it is the primary consumption surface for BNII — behavioural network intelligence drawn from consented, in-market consumer signal across emerging markets.

## One-line thesis

Anyone can rent a frontier model. No one can rent the position it reasons from.

## What exists now

| Layer | Status |
| --- | --- |
| Chat + streaming | Live |
| Deep Research briefs | Live (prompt contract + UI parser) |
| Independent web research | Live (search → fetch → ground → answer) |
| Multi-model routing (Nano / Mini / Max) | Live |
| Export (PDF, DOCX, PPTX, XLSX, CSV, JSON) | Live (client-side) |
| Thread history & share links | Live (browser-local) |
| Auth, credits, metering ledger | Not built |
| Governed intelligence APIs | Not built |
| Bring-your-own-data / run-on-infra | Not built |
