## Brief section headings (stable)

Deep Research parsers depend on these markdown labels remaining stable:

`Title` · `Executive Summary` · `Key Findings` · `Detailed Analysis` · `Areas of Uncertainty` · `Sources`

Legacy aliases still accepted: `Headline`, `Analysis`.

## Future contract principle

When credits land, the **audit event schema** becomes a public contract. Design it once; reuse for billing, compliance, and customer-facing usage statements.
