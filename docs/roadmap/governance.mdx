# Governance

Governance is not a compliance appendix. It is the commercial enabler for deeper surfaces.

## Three gates

| Gate | Meaning |
| --- | --- |
| **Consent** | Signal enters the intelligence layer only under partner / user consent terms |
| **k-floor** | Aggregates release only above minimum cohort size; no re-identification paths |
| **Audit + meter** | Every query / pull / job emits a durable event used for credibility **and** billing |

## Deepest surface rule

On **Run on our infra**, raw behavioural signal never leaves the boundary. Clients receive intelligence output only. This is both the strongest governance posture and the deepest lock-in — sell it as one story.

## Aria today

Aria does not yet implement these gates. Documentation states them so implementation cannot invent per-feature exceptions later.
